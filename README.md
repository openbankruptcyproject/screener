# screener.openbankruptcyproject.org

**§ 523(a)(8) AP Eligibility Self-Screener**

Interactive educational self-assessment tool that walks the user through Brunner-test factors and outputs a non-binding factor-by-factor educational summary. Pure HTML+JS, no data is collected or saved.

Live site: <https://screener.openbankruptcyproject.org/>

Maintained by the [Open Bankruptcy Project](https://openbankruptcyproject.org/), a 501(c)(3) public charity (EIN 41-5159631).
